Installation

To install the package, you can use pip:

pip install nyancat

Usage

Load a color map using the id:

from nyancat import NyanCat

nyancat = NyanCat()
cmap = nyancat.load_cmap('dafed9')


Load a random color map:

from nyancat import NyanCat

nyancat = NyanCat()
cmap = nyancat.load_cmap('random')
